Who We Are

CineList is a web application developed to provide a seamless movie catalog management experience. This project was built using front-end development best practices, focusing on usability and responsive design through the Bootstrap framework.

The main goal is to demonstrate the integration of dynamic interfaces with listing systems, providing the user with a lightweight, functional, and aesthetically pleasing tool for registering and viewing cinematographic titles.
